What it is:
- A single-page web app (HTML/CSS/JS) that generates a study timetable from user inputs.
- No backend required. Runs fully in-browser.

How to use:
1. Open index.html in a browser.
2. Enter subjects (one per line). Use format:
   Subject: topic1|topic2|topic3
   or just: Subject
3. Choose exam date (must be future) and hours available per day.
4. Click Generate Plan.
5. Print or save the generated plan.

- index.html
- style.css
- script.js
